## 七阶段路线图

1. **工程基建** ✅ — Git init、.gitignore、requirements.txt、README.md、Initial Commit
2. **需求文档化** ✅ — PRD 模板创建 + Jasmine 撰写初稿 + 修订，已 commit
3. **代码架构重构** — 模块化拆分（MVC 结构）
4. **功能迭代** — Sprint 1(任务增强)、Sprint 2(数据体验)、Sprint 3(统计看板)

## 当前进度

- 阶段 1、2 已完成
- PRD v0.2 已定稿，v0.2 范围确认为：
  - **Must:** 日程删除、统计看板、当日截止步骤汇总
