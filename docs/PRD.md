## 1. 背景与目标

<!-- 引导：用 3-5 句话回答 ——

这个项目缘起我个人的时间管理需要。因为我目前没有在市面上看到一款功能简单明了、使用方便、界面干净、功能可自定义的轻量化日程管理软件，又有记录日常学习、工作任务的需要。

目前的版本（v0.1）已经可以分类记录短期的每日任务与长期项目，其中长期项目可以添加具体步骤并设置截止时间；可以在日历界面根据已有日程的截止日期、以不同的颜色区分显示其紧迫程度。每个日程的已/未完成状态、截止时间和日程的具体内容可以自定义、随时修改。

新一版（v0.2）准备完成以下更新。一是添加一个数据统计看板。看板可以总结整理月度/季度的日程完成情况，将其做成趋势图、并计算日程的完成率。二是明确用户写入数据的存储位置，可以将这些数据输出为csv/excel格式的文件、便于保存。三是增加删除日程的功能。四是将长期项目中的截至日期为当日的步骤同样显示在当日要完成的页面，一并提醒用户。

## 5. 功能详述

<!-- 引导：对每个 Must 和 Should 功能，展开描述具体的交互流程。
  不需要写代码逻辑，重点是"用户看到什么、做什么、系统怎么响应"。
  格式建议：一个功能 = 一段描述 + 关键交互步骤 + 边界情况

### 5.1 日程删除

- **交互流程：** 在每日任务和长期项目页面底部各有一个"删除任务"/"删除项目"按钮。用户先点击列表中的条目将其选中，再点击底部的删除按钮，弹出确认窗口"确定要删除「XXX」吗？此操作不可恢复。"，用户确认后删除。长期项目中的步骤通过双击进入编辑弹窗，在弹窗内点击"删除"按钮并确认后删除。
- **边界情况：** 未选中任何条目时点击删除按钮，弹出提示"请先选中一个条目"。

### 5.2 数据统计看板

- **交互流程：** 用户点击”数据统计”标签页进入统计视图。页面包含一张折线图，纵坐标为日程数量，横坐标为时间，用蓝色和绿色两条折线分别表示”全部日程”与”已完成”数量，每个节点旁标注具体数字。图表下方有三个时间范围切换按钮（”过去4周””过去3个月””过去12个月”），点击后图表内容随之更新。图表上方显示汇总文字（总计 / 已完成 / 完成率）。
- **边界情况：** 若所选时间范围内无数据，图表显示为空白或只有零值轴。时间范围最多支持过去 12 个月。

### 5.3 当日日程汇总

- **交互流程：** 目前每日任务的页面可添加和显示当日的任务。新一版的改动希望用户在添加长期项目之步骤时，若有步骤的截止日期为当日，则一并显示在每日任务的页面，并且可以在该栏当中用加粗字体备注显示该任务属于某个长期项目。此时每日任务的排列顺序按创建该日程的时间（若是长期项目的步骤则以该步骤的创建时间为准）倒序排列，也即最新创建的日程在最上面。
- **边界情况：** 暂无。

### 5.4 数据导出

- **交互流程：** 
  在标签栏右侧增加一个"导出数据"按钮。点击后弹出窗口，用户通过两个日期输入框（起始日期 + 结束日期）选择导出范围，另有快捷按钮"本月""近3个月""全部"。确认后弹出文件保存对话框，用户选择保存位置和文件名。导出格式为 CSV，包含：任务/步骤名称、所属项目（如有）、截止日期、完成状态。
- **边界情况：** 若选中的时间范围内无数据，提示用户"所选范围无日程记录"；日期范围为过去/未来 1 年内。

## 6. 验收标准

### 日常操作

- [x] **A-1 切换状态**：点击任务/步骤左侧状态标记（✓/✗），状态立即切换，刷新后保持
- [x] **A-2 双击编辑**：双击任务/步骤弹出编辑窗口，预填当前内容（步骤含截止日期），修改后点”保存修改”成功更新
- [x] **A-3 添加任务/步骤**：点击”添加任务”或”添加步骤”按钮，输入内容后出现在列表中；步骤可填写可选的截止日期

### 删除操作

- [x] **B-1 删除任务**：选中每日任务列表中的条目，点击底部”删除任务”按钮，弹出确认窗口，确认后条目从列表消失
- [x] **B-2 删除步骤**：双击步骤进入编辑弹窗，点击”删除”按钮，确认后步骤从项目详情中消失
- [x] **B-3 删除项目**：选中长期项目列表中的项目，点击底部”删除项目”按钮，确认后项目及其所有步骤被删除
- [x] **B-4 防误删**：未选中条目时点击删除按钮 → 提示”请先选中一个条目”；删除前均有确认弹窗

### DDL 汇总与自动同步

- [x] **C-1 当日截止步骤**：长期项目中截止日期为当日的步骤，自动出现在”每日任务”页面，标注所属项目名（如”【学期论文写作】完成文献综述”）
- [x] **C-2 项目自动完成**：当项目的所有步骤都完成后，项目自动标记为 ✓；当某个步骤被取消完成、导致步骤未全部完成时，项目自动回到 ✗

### 数据统计

- [x] **D-1 图表展示**：进入”数据统计”标签页，显示折线图（蓝色=全部日程，绿色=已完成），节点标有数字
- [x] **D-2 范围切换**：点击”过去4周””过去3个月””过去12个月”按钮，图表数据相应更新
- [x] **D-3 汇总信息**：图表上方显示”总计 / 已完成 / 完成率”汇总文字

### 数据导出

- [x] **E-1 导出窗口**：点击标签栏右侧”导出数据”按钮，弹出窗口，包含起始日期、结束日期两个输入框和”本月””近3个月””全部”快捷按钮
- [x] **E-2 导出文件**：选择范围后确认，弹出文件保存对话框，默认文件名含日期范围，保存后生成 UTF-8 编码 CSV 文件
- [x] **E-3 CSV 内容**：导出文件包含列：类型（任务/步骤）、名称、所属项目、截止日期、完成状态；日期范围内的数据正确完整
- [x] **E-4 空数据提示**：所选范围无数据时，提示”所选范围内无数据”

### 非功能

- [x] **F-1 数据持久化**：关闭程序后重新打开，之前添加的所有任务和项目仍然存在，状态不变
- [x] **F-2 异常处理**：数据文件损坏（手动修改 JSON 格式错误）时，程序不崩溃，给出明确提示后以空数据启动
